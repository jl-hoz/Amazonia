# Amazonia [![Build Status](https://travis-ci.com/joseluishozg/Amazonia.svg?token=LEERDZhZNp4DPaTBmFXL&branch=master)](https://travis-ci.com/joseluishozg/Amazonia)

Amazonia is a software that manages products and categories of a store.

This software was created as a university practice. When the semester is over this repository will be archived and also will become private.

## Installation

To install the software is mandatory to have Gradle build tool.

Unix/Unix-like users execute build as:
```
./gradlew
```

Windows users execute build as:
```
gradlew.bat
```
